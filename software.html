Game Engines

Game Engines are like normal software engines but made specifically
for game creation. They provide things like libraries of support programs
that are standard in all games. They also provide level editors and a
framework for the game itself. Game engines have a feature called
platform abstraction whicha allows developers to make their games on
differnt platforms such as Xbox and Playstation.

Unity Engine

The Unity Engine is one of the mainly used game engines in game
development. It was made by Unity Technologies and originally released in 2005 and quickly became very popular
and went through a series of updates and changes from now to the present
Unity has gone through a total of 5 upgrades starting with the original
Unity then Unity 2.0 - Unity 5.0. Eventually they dropped the numbering system
and just kept the name Unity universal.

Unreal Engine

The Unreal Engine is another very popular engine in the gameing industry.
It was one of the first 3D computer engine used in game making. It was developed
by Time Sweeny, who is also the founder of Epic Games and released in 1998 with
its first-person shooter game "Unreal". The engine itself has gone through a
series of changes and updates but unlike the Unity engine it kept its numbering
system. Currently the Unreal Engine is on Unreal 5.
